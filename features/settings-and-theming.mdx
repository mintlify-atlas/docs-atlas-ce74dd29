## Opening Settings

Tap the **Settings** tab in the bottom navigation bar to open the settings screen. Settings are organized into three sections: **Account**, **Appearance**, and **App**.

---

## Account

The **Account** section shows your profile information and provides a way to sign out.

- **Display name** — your name as derived from your login email, shown at the top of the card.
- **Username** — your full login email address, shown below your display name.
- **Logout** — tap to sign out of your account. See [Authentication](/features/authentication) for details on the logout flow.

---

## Appearance

The **Appearance** section lets you control how the app looks.

### Theme mode

Tap the **Theme Mode** row to cycle through the three available modes:

| Mode | Description |
|---|---|
| **System default** | Follows your device's system-wide light/dark setting |
| **Light mode** | Always uses the light color scheme |
| **Dark mode** | Always uses the dark color scheme |

Each tap advances to the next mode in the cycle: System → Light → Dark → System.

### Theme color

The app uses a customizable accent color to style buttons, icons, and highlights. The default color is `#4b5c92` (a deep blue-indigo).

<Steps>
  <Step title="Open color picker">
    Tap the **Theme Color** row in the **Appearance** section. A color picker dialog opens showing the current color.
  </Step>
  <Step title="Choose a color">
    Use the color picker to select any color you like. A preview of the selected color updates as you drag.
  </Step>
  <Step title="Confirm">
    Tap **Select** to apply the new color. Tap **Cancel** to discard the change and keep the existing color.
  </Step>
</Steps>

<Note>
  Your chosen theme color is saved persistently on your device. It will still be applied the next time you open the app.
</Note>

## App

The **App** section provides update and version information.

- **Check for Updates** — tap to check whether a newer version of Openly is available. If an update is found, you will be prompted to install it. If you are already on the latest version, a dialog confirms that no update is needed.
- **Version** — displays the current installed version number of the app.
